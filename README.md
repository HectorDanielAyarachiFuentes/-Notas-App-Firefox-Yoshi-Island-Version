# Notas Pro - Yoshi's Island Edition 🦖🐾

Una extensión vibrante y artesanal para **Mozilla Firefox** que permite gestionar tus notas con una estética única inspirada en el arte de Yoshi's Island, combinando un diseño de libreta dibujada a mano con funciones potentes de OCR y sincronización.

## ✨ Características

- 📝 **Estética Yoshi's Island**: Una interfaz premium dibujada a mano, con texturas de papel, efectos de crayón y animaciones dinámicas.
- 🔍 **OCR Selectivo**: Extrae texto de cualquier parte de la pantalla directamente a tus notas (¡Incluso traduce el texto extraído!).
- 📝 **Gestión de Notas**: Crea, edita y organiza tus notas con una experiencia táctil y visualmente impactante.
- ☁️ **Sincronización Local y Cloud**: Gestión eficiente de datos optimizada para Firefox.
- 🌓 **Diseño Adaptativo**: Se ajusta perfectamente al panel lateral o al popup de Firefox.

## 🚀 Instalación en Firefox (Modo Desarrollador)

1. Descarga o clona este repositorio.
2. Abre **Firefox** y escribe `about:debugging` en la barra de direcciones.
3. Haz clic en **"Este Firefox"** (This Firefox) en el menú de la izquierda.
4. Haz clic en el botón **"Cargar complemento temporal..."** (Load Temporary Add-on...).
5. Navega hasta la carpeta del proyecto y selecciona el archivo `manifest.json`.

## 🛠️ Tecnologías

- HTML5 / CSS3 (Variables HSL, Filtros SVG, Animaciones)
- JavaScript (WebExtensions API para Firefox)
- Integración de OCR avanzada
- Diseño UI/UX artesanal (Hand-drawn design system)
